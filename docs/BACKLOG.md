## [BACKLOG-001] 홍보 카드 만들기

**개요**
생성된 음식 이미지 위에 메뉴명, 홍보 문구, 가격 등을 얹어 인스타·당근·스마트플레이스용 홍보 이미지를 만드는 기능.

**사용자 흐름**
1. 기존 AI 생성 결과 이미지를 베이스로 사용한다.
2. 사용자가 아래 항목을 직접 입력한다.
   - 메뉴명
   - 설명 문구 (재료, 특징 등)
   - 가격 / 이벤트 문구 (선택)
3. 앱이 텍스트를 이미지 위에 합성해 완성 카드를 출력한다.
4. 사용자가 완성본을 다운로드한다.

**구현 단계**

| 단계 | 방식 | 비고 |
|------|------|------|
| 초기 (안전) | 앱 내 HTML/CSS 오버레이 또는 canvas API로 텍스트 합성 | 서버 비용 없음, 한글 자유 |
| 실험 | GPT 이미지 생성 프롬프트에 한글 문구 포함 | 한글 렌더링 불안정, 비용 발생 |
| 상용화 | 사용자가 문구 위치·폰트·색상 수정 가능한 에디터 | 고도화 필요 |

**출력 비율**
- 1:1 (인스타 피드, 배달앱 대표사진)
- 4:5 (인스타 세로형)
- 9:16 (인스타 스토리, 당근 스토리)

**기술 고려사항**
- 초기 버전: `html2canvas` 또는 브라우저 Canvas API
- 폰트: 한글 웹폰트 로딩 타이밍 주의
- 이미지 출력: `canvas.toBlob()` → 다운로드

**우선순위 판단 기준**
- 기존 생성 기능이 안정화된 뒤 착수
- 사용자 피드백에서 "문구 넣기" 요청이 반복되면 1순위로 올림
